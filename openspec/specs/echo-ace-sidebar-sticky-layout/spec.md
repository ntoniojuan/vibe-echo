## ADDED Requirements

### Requirement: Sticky teammate sidebar on ACE evaluation steps

For ACE evaluation steps (Aptitude, Character, Effectiveness) where the layout presents the teammate history sidebar alongside the main form, the sidebar container SHALL be sticky to the top of the scrolling region so users can reference history while editing long fields.

#### Scenario: User scrolls main ACE content

- **WHEN** the user scrolls vertically within the main ACE content area on side-by-side layouts
- **THEN** the teammate history sidebar SHALL remain visible at the top of the viewport per sticky positioning rules
