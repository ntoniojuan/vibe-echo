## ADDED Requirements

### Requirement: Dashboard inbox card places primary action on ACE pillar row

On dashboard inbox cards, the primary action control (e.g. **Open** or **Continue**) SHALL appear on the **same row** as the Head / Heart / Hands (ACE) pillar indicators, subject to responsive wrapping, so the card is less vertically bulky than a separate full-width action row below the pills.

#### Scenario: Card with primary action

- **WHEN** an inbox card displays ACE pillar pills and a primary action is applicable
- **THEN** the primary action SHALL be laid out on the same row as the ACE pills (with acceptable wrap on very narrow widths) and SHALL NOT occupy a separate block row solely for vertical stacking below the pills in the default desktop / tablet layout
