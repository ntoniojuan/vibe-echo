## ADDED Requirements

### Requirement: Profile row shows a leading icon

The desktop sidebar footer Profile link SHALL display a leading icon before the “Profile” label.

#### Scenario: Profile icon matches primary nav styling

- **WHEN** the sidebar footer Profile row is rendered on desktop
- **THEN** a leading icon SHALL appear before the label
- **AND** the icon SHALL match the size, stroke weight, and spacing (`gap` alignment) used for primary sidebar navigation items

### Requirement: Logout control shows a leading icon

The desktop sidebar footer Logout control SHALL display a leading icon before the “Logout” label.

#### Scenario: Logout icon matches primary nav styling

- **WHEN** the sidebar footer Logout button is rendered on desktop
- **THEN** a leading icon SHALL appear before the label
- **AND** the icon SHALL match the size, stroke weight, and spacing used for primary sidebar navigation items
